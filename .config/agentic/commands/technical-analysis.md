# Technical Analysis

Produce a technical analysis document using the `technical_analysis` skill instructions.

## When to use

- `/technical-analysis`
- User provides a task link (Phabricator, Jira, GitHub issue, etc.) and asks for planning or analysis.
- User asks "how long would this take", "what would need to change", or "where do I start".
- User asks for an architecture review or refactoring candidates.
