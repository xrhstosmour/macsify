# Caveman

Activate caveman mode. Apply the caveman skill rules to all subsequent responses until user says "stop caveman" or "normal mode".

## When to use

- `/caveman`
- User says "caveman mode", "talk like caveman", "use caveman", "less tokens", "save tokens", or "be brief".

See `skills/caveman/SKILL.md` for the full rule set.
