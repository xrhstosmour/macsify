# Create Phabricator Task

Create a Phabricator task using the `create_phabricator_task` skill instructions.

Only trigger this command when the user explicitly mentions Phabricator or "phab": `create phab task`, `create phab ticket`, `create phab issue`, `create Phabricator task`, `create Phabricator ticket`, `create Phabricator issue`, or `/create-phabricator-task`.
